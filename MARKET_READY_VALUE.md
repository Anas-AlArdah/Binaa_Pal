# Binaa Pal - جاهزية تجارية وتسعير

آخر تحديث: 2026-06-07

## التموضع

Binaa Pal هو Marketplace محلي للخدمات الحرفية في فلسطين. القيمة ليست في عرض أسماء الحرفيين فقط، بل في تحويل رحلة العميل إلى مسار منظم:

- بحث حسب الصنعة أو اسم الحرفي.
- مقارنة ملفات الحرفيين من حيث المدينة، المهارات، الصور، التقييمات، الأسعار، والتوفر.
- إرسال طلب خدمة ومتابعته.
- لوحة للحرفي لإدارة الملف والطلبات.
- لوحة Admin لإدارة المنصة.

## لماذا يمكن عرضه كمنتج بقيمة 5000 دولار

هذه القيمة تكون واقعية عند بيع المشروع كنسخة MVP قابلة للتخصيص والإطلاق، لا كفكرة فقط. عناصر القيمة الحالية:

- واجهة React عربية RTL مع تجربة بحث وملفات حرفيين وصفحات صنعات.
- API مبني بـ Node.js وExpress وSequelize مع مصادقة JWT وتسجيل Google اختياري.
- نماذج بيانات للعمال، المهارات، الطلبات، التقييمات، المشاريع، الصور، التوفر، والعروض.
- حماية ملكية أساسية في Middleware تمنع المستخدم من تعديل ملفات غيره.
- طبقات أمان إنتاجية مضافة: security headers، rate limiting، وفحص أسرار الإنتاج.
- تجهيز نشر موجود لـ Netlify وRailway.
- صفحة رئيسية محسنة تجارياً تعرض الثقة، خطوات الاستخدام، مناطق الخدمة، ودعوة الحرفيين للتسجيل.

## إشارات سوقية مختصرة

تمت مراجعة أمثلة قريبة في السوق الفلسطيني والإقليمي:

- RAHEQ-ALQUDS يعرض خدمات صيانة منزلية في فلسطين مع رسائل ثقة وخدمات شائعة: https://www.iphontel.com/en/
- Softahs Palestine وPallancer يثبتان وجود طلب على منصات خدمات/عمل حر فلسطينية: https://softahs.com/ و https://pallancer.com.ps/
- Hirly يعرض تموضعاً فلسطينياً حول توظيف المهنيين والثقة والتخصصات: https://www.hirly.net/
- KABLAY مثال إقليمي قريب لفكرة Marketplace صيانة منزلية حسب الفئات: https://kablay.co.il/en/about

الفرصة لـ Binaa Pal أن يتمركز بشكل أضيق من منصات العمل الحر العامة: خدمات حرفية محلية منزلية مع ملفات ثقة وطلب خدمة سريع.

## نموذج دخل قابل للإضافة

- اشتراك شهري للحرفي المميز.
- عمولة على الطلب المكتمل.
- باقات ظهور أعلى داخل نتائج البحث.
- صفحة موثقة للحرفي تشمل شارة تحقق وصور أعمال أكثر.
- لوحة مؤسسات لشركات المقاولات أو الصيانة.

## ما تم رفعه في هذه النسخة

- تحسين الصفحة الرئيسية كواجهة بيع: رسالة أوضح، مؤشرات قيمة، خطوات استخدام، ثقة، تغطية مدن، وCTA للحرفيين.
- تشغيل بحث الاسم فعلياً في الـ API عند إرسال `name` من الواجهة.
- إضافة `helmet` و`express-rate-limit` لحماية API ومحاولات تسجيل الدخول.
- إضافة فحص إنتاج يمنع التشغيل بـ `JWT_SECRET` أو كلمة Admin افتراضية.
- تحسين SEO وPWA metadata في `index.html` و`manifest.json`.

## خطوات مقترحة قبل البيع النهائي

- تشغيل Seed ببيانات حقيقية أو شبه حقيقية لـ 20-40 حرفياً.
- تصوير Demo قصير مدته 60-90 ثانية: بحث، عرض ملف، إرسال طلب، دخول حرفي، تحديث طلب.
- إضافة صفحة "من نحن" و"سياسة الخصوصية" قبل الإطلاق العام.
- إضافة رفع صور حقيقي عبر Cloudinary أو S3 بدلاً من base64 أو روابط يدوية.
- إضافة اختبارات API للمصادقة والبحث وتعديل الملفات.
